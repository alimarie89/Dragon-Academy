# Dragon Academy

Student knowledge map and resource hub for Dragon Academy — a personal growth and development program exploring power, embodiment, and relational range.

## Project Structure

```
dragon-academy/
├── index.html              # Main HTML file
├── css/
│   └── styles.css         # All styling
├── js/
│   └── script.js          # JavaScript functionality
├── images/
│   ├── carousel/          # Carousel images
│   ├── slides/            # Full-width slide images
│   └── team/              # Team member photos
└── README.md              # This file
```

## Deployment

This site is deployed via Netlify, connected to this GitHub repository. Any push to the `main` branch will trigger an automatic redeploy.

## Development

To work on this project locally:
1. Clone this repository
2. Open `index.html` in a browser — no build step required
3. Edit files and push to `main` to deploy
